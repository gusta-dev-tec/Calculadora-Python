# Calculadora Python

Projeto de calculadora feito em Python no terminal para praticar lógica de programação e funções.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Potenciação
- Histórico de usuarios

## Tecnologias

- Python 3

## Como executar

```bash
python calculadora.py
````

## Conceitos praticados

* Funções (`def`)
* Condições (`if / elif / else`)
* Laço de repetição (`while`)
* Listas
* Tratamento de erros (`try / except`)

Projeto criado durante meus estudos para evoluir em Python e desenvolver sistemas simples.
